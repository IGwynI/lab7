## Задача 1

### Текст задачи

Файл содержит сведения об игрушках: название игрушки, ее стоимость в рублях и возрастные
границы (например, игрушка может предназначаться для детей от двух до пяти лет). Получить
название игрушек, цена которых не превышает k руб. и которые подходят детям 5 лет.

### Алгоритм решения

Заполнение файла: 
Создать пустой список объектов Toy; 
Для каждого элемента:
Сгенерировать случайное название (из заданного набора + число). 
Сгенерировать случайную цену в диапазоне 100–5000 руб. 
Сгенерировать минимальный возраст (0–9 лет) и максимальный возраст (minAge + случайное 0–9, но не более 18). 
Создать объект Toy и добавить в список; 
Сериализовать список в XML-строку с помощью XmlSerializer; 
Преобразовать XML-строку в массив байт; 
Записать в бинарный файл: сначала длину массива байт (4 байта), затем сам массив.
Чтение файла: 
Открыть файл для бинарного чтения; 
Считать первые 4 байта - длину массива XML; 
Считать указанное количество байт и преобразовать их в строку; 
Десериализовать XML-строку обратно в список объектов Toy; 
Для каждого объекта проверить условия: Price <= maxPrice, MinAge <= 5 <= MaxAge; 
Вывести подходящие игрушки или сообщение об их отсутствии.

count == n -> «посмотрели все»

## Задача 1

### Текст задачи

Файл содержит текст на русском языке. Напечатать в алфавитном порядке все звонкие
согласные буквы, которые входят более чем в одно слово.

### Алгоритм решения

Создать множество звонких согласных: б, в, г, д, ж, з, й, л, м, н, р; 
Создать словарь Dictionary<char, int> для подсчёта количества слов, в которых встречается каждая буква; 
Открыть файл и читать построчно: 
Разбить строку на слова; 
Для каждого слова: 
Создать HashSet<char> для уникальных звонких согласных этого слова. 
Для каждого символа слова: если он входит в множество звонких - добавить в HashSet. 
Для каждой буквы из этого множества увеличить соответствующий счётчик в словаре (если буква встречается впервые – добавить с начальным значением 1). 
После обработки всего файла отобрать буквы, у которых счётчик > 1;

# Задание 10

## Задача 1

### Текст задачи

На вход программе подаются сведения о телефонах всех сотрудников некоторого учреждения. В
первой строке сообщается количество сотрудников N, каждая из следующих N строк имеет
следующий формат:
<Фамилия><Инициалы><телефон>
где <Фамилия> – строка, состоящая не более чем из 20 символов, <Инициалы> - строка, состоящая
не более чем из 4-х символов (буква, точка, буква, точка), <телефон> – семизначный номер, 3-я и
4, я, а также 5-я и 6-я цифры которого разделены символом «–». <Фамилия> и <Инициалы>, а
также <Инициалы> и <телефон> разделены одним пробелом. Пример входной строки:
Иванов П.С. 555-66-77
Сотрудники одного подразделения имеют один и тот же номер телефона. Номера телефонов в
учреждении отличаются только двумя последними цифрами. Требуется написать эффективную
программу, которая будет выводить на экран информацию, сколько в среднем сотрудников
работает в одном подразделении данного учреждения.

### Алгоритм решения

Создать словарь Dictionary<string, int>: ключ – код подразделения (5 цифр), значение – количество сотрудников; 
Открыть файл и читать построчно: 
Первая строка - ожидаемое количество сотрудников. 
Пропускать пустые строки; 
Для каждой последующей строки:
Найти последний пробел - после него следует номер телефона. 
Извлечь номер, удалить дефисы. 
Проверить, что номер содержит ровно 7 цифр. 
Взять первые 5 цифр как код подразделения. 
Увеличить счётчик для этого кода в словаре (если код новый – создать запись со значением 1). 
Если словарь пуст - вернуть 0; 
Разделить сумму на количество ключей (число подразделений) и вернуть результат как double.
